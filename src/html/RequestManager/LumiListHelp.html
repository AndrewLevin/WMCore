Run/Lumi lists are accepted in any of the following formats:
  • runs
    [1,2,5]
  • "Compact form"
    {
  '1': [[1, 33], [35, 35], [37, 47], [49, 75], [77, 130], [133, 136]],
  '2':[[1,45],[50,80]]
}
  • runLumiDict
    {
  '1': [1,2,3,4,6,7,8,9,10],
  '2': [1,4,5,20]
}
  • runLumiPairs
    [[1,1], [1,2],[1,4], [2,1], [2,5], [1,10]]
